## 升级说明

v0.3.7 不包含数据库结构变更，可以从 v0.3.6 直接升级。

## 主要变化

- 报告阅读页和 Chat 现在会把 `mermaid` 代码块渲染为图表，包括研报中使用的
  `xychart-beta`。
- 图表按需加载，并随门户的明暗主题重新渲染；不含图表的页面不会下载 Mermaid。
- 流式 Chat 会等待 Mermaid 围栏闭合后再渲染。语法错误、未完成或过大的图表会安全降级为
  原始代码块，不会影响报告或对话的其他内容。
- Mermaid 使用严格安全模式，生成的 SVG 在插入页面前会再次清洗，并限制正文大小和图边数量。
- 修正本机缺少 PDF 引擎时已经失效的 Homebrew 安装提示。

## 已知限制

- Markdown 下载会继续完整保留 Mermaid 源码。
- 本版本尚未在 PDF 和按日 ZIP 内渲染 Mermaid；这两个出口仍显示代码块。服务端 PDF 渲染方案
  仍需通过生产 wkhtmltopdf 的内联 SVG 探针后才能启用。
- Mermaid `xychart-beta` 不支持双 Y 轴。价格与 RSI/KD 等不同量纲指标应拆成上下两个图表。
