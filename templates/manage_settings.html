新建令牌

入库（Dify 工作流末尾 HTTP 节点）

POST /api/reports · Header Authorization: Bearer <令牌> · JSON：
{"run_id","symbol","date","kind","subtype","title","body_md"} （kind 大类不填按 subtype 推断；run_id 归组一次生成的多篇）。

查 / 搜（Dify 复核历史假设与跟踪事项）

先探清单（有哪些、多久前、是否有）：
GET /api/reports/manifest?symbol=300750 → {name,total,dates:[{date,count,kinds}],kinds,subtypes}
再查 / 搜（symbol 或 q 至少给一个，q 搜标题+正文）：
GET /api/reports?symbol=300750&q=关键字&kind=&subtype=&since=&until=&limit=20&with_body=1
取单篇正文：GET /api/report?uid=... → 含 body_md / body_html
全部走同一 Bearer 令牌（作用域含 query）。
